# Blinken

A macOS menu bar utility that resurrects the ambient hardware-status indicators of pre-2010 personal computers — starting with the HDD activity LED.

> 🚧 **In development.** v1.0 spec lives in [`PRD.md`](./PRD.md). Full README to follow once the app builds and runs.
