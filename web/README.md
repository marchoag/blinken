# The website lives in the Axiomic repo

The page at **https://labs.axiomic.ai/blinken** is built and deployed from the
Axiomic-LLC repo, at `labs/blinken/` — not from here.

This folder previously held a copy of that site. It was the original template,
it stopped being deployed, and it quietly drifted out of sync, so edits made
here looked correct and changed nothing. It has been removed to leave exactly
one place the page can be edited.

To change the site, edit `labs/blinken/` in the Axiomic-LLC repo and push;
Vercel deploys from `main`.
